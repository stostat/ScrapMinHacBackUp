# INSTALACION DE MODULOS EN PYTHON3

------------


Instalamos con el siguiente comando los paquetes necesarios:

------------

- pip3 install **pandas**
- pip3 install **re**
- pip3 install  **pytesseract** 
- pip3 install **pdf2image**
- pip3 install pillow **(PIL)**

------------

## Comando para correr el  *file.py*:

------------
- python3 file.py

------------
